Diffrence between HTML and CSS
HTML CSS
Defines the structure and content of a webpage Defines the style, design, and layout of a webpage
Acts as the skeleton (headings, paragraphs, images, links, forms, etc.) Acts as the skin/clothing (colors, fonts, spacing, positioning, animations, etc.)
Markup language Style sheet language
Can exist alone (a plain webpage without style) Cannot work alone, always needs HTML or XML to style
